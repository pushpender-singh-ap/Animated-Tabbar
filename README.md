## How To Use
```
  - git clone https://github.com/pushpender-singh-ap/Animated-Tabbar.git
  - cd Tabbar
  - npm install
  - expo start
```

## Software Used
```
  - VSCode
  - Expo-cli
```
## Dependencies Install
```
  - npm install d3
  - npm install d3-interpolate-path
  - npm install d3-path
  - npm install d3-shape
  - npm install react-native-svg
```
